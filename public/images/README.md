# Images Folder

Letakkan file gambar Anda di sini.

## Untuk Logo/Icon TELATEN

- Nama file yang disarankan: `logo.png`, `logo.svg`, atau `heart-icon.png`
- Format yang didukung: PNG, SVG, JPG, WEBP
- Ukuran yang disarankan: 40x40px hingga 100x100px untuk logo

## Cara menggunakan

File yang diletakkan di folder `public/` dapat diakses langsung dengan path `/images/namafile.ext`

Contoh:

- File: `public/images/logo.png`
- Akses di kode: `/images/logo.png`
